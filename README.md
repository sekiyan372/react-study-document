## 勉強会の目的
- React を知り、各々のフロントエンドの技術選定の選択肢の一つとなること
- Web フロントエンドの楽しさを知ること
- 他プロジェクトの人と交流を深めること

## 参加するにあたって知っておいてほしい前提知識
- プログラミングの基本的な習熟

- CSS の基本的な習熟（自分で簡単なサイトが作成できるレベル）
- JavaScript の基本的な習熟（ES2015 (ES6) 以降の記法）
- Git のコマンドによる基本的な操作
