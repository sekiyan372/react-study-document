## 勉強会の目的
- Reactを知り、各々のフロントエンドの技術選定の選択肢の一つとなること
- Webフロントエンドの楽しさを知ること
- 他プロジェクトの人と交流を深めること

## 参加するにあたって知っておいてほしい前提知識
- プログラミングの基本的な習熟
- HTMLの基本的な習熟（自分で簡単なサイトが作成できるレベル）
- CSSの基本的な習熟（自分で簡単なサイトが作成できるレベル）
- JavaScriptの基本的な習熟（ES2015(ES6)以降の記法）
- Gitのコマンドによる基本的な操作
